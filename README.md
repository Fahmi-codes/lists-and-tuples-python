# Lists and Tuples in Python

This project explores two of the most important data structures in Python: Lists and Tuples.  
The notebook contains practical examples, operations, and explanations to help understand how these structures work in real Python programs.

---

## 📌 Topics Covered

### 🔹 Python Lists
- Creating lists
- Accessing elements
- List slicing
- Adding items (`append`, `insert`, `extend`)
- Removing items (`remove`, `pop`, `clear`)
- Updating list values
- Looping through lists
- Nested lists
- List methods and operations

### 🔹 Python Tuples
- Creating tuples
- Accessing tuple elements
- Tuple slicing
- Tuple unpacking
- Immutable nature of tuples
- Tuple methods (`count`, `index`)
- Nested tuples

### 🔹 Lists vs Tuples
- Mutability
- Performance differences
- Use cases
- Memory efficiency
